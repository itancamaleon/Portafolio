Skölays

Skölays es una plataforma social en línea inspirada en la simplicidad de WhatsApp, pero con funcionalidades avanzadas que la convierten en una experiencia mucho más completa y personalizable. Está pensada para ser ligera, intuitiva y con una interfaz amigable.

Características principales

  • Inicio de sesión seguro con autenticación por correo electrónico.
  • Gestión de amigos: enviar, aceptar y rechazar solicitudes de amistad.
  • Chats en tiempo real con soporte para mensajes de texto e imágenes.
  • Grupos dinámicos: crea, administra y conversa con múltiples usuarios.
  • Perfil personalizable: cambia tu foto de perfil y ajusta tu información.
  • Diseño amigable: interfaz clara, moderna y fácil de usar.

Tecnologías utilizadas

Backend con Flask y Socket.IO para comunicación en tiempo real.
Base de datos SQLite para gestionar usuarios, amistades y mensajes.
Frontend con HTML, CSS y JavaScript para una experiencia fluida.
Sistema de verificación por correo electrónico para mayor seguridad.

Cubotrix

Cubotrix es un juego arcade diseñado con HTML, CSS y JavaScript.
La mecánica es sencilla pero desafiante: controla un cubo morado que debe esquivar obstáculos rojos y alcanzar la meta verde.

En su versión completa el juego cuenta con más de 10 niveles y 3 dificultades, pero aquí podrás probar un nivel de demostración. Si completas el nivel o pierdes, el juego se reiniciará automáticamente.

Este proyecto demuestra el uso de canvas, manejo de colisiones, detección de eventos de teclado y un sistema de reinicio dinámico para la experiencia arcade.

Wiki de Cubos

Wiki de Cubos es una plataforma en línea diseñada para los entusiastas del cubo de Rubik y otros rompecabezas similares. Su objetivo es ofrecer una experiencia informativa, organizada y colaborativa donde los usuarios pueden aprender, consultar y compartir soluciones, métodos y curiosidades del mundo del speedcubing.

Características principales

  • Artículos completos sobre cubos 2x2, 3x3, 4x4, hasta categorías avanzadas.
  • Guías ilustradas con algoritmos paso a paso para principiantes y expertos.
  • Sección de récords y curiosidades sobre los cubos más rápidos y raros.
  • Diseño intuitivo: navegación clara y fácil para acceder a la información.
  • Contenido colaborativo: posibilidad de expansión con más métodos y variantes.

Tecnologías utilizadas

Desarrollada con HTML, CSS y JavaScript para lograr una interfaz accesible y rápida.
Información organizada en formato estilo wiki, pensada para crecer y actualizarse con facilidad.
Optimización para lectura clara y compatibilidad con distintos dispositivos.

Consola Retro

Consola portátil construida desde cero con ESP32 y pantalla OLED 128x64, controles físicos y batería recargable. Incluye 9 juegos clásicos,
almacenamiento de puntajes y un panel web para consultar estadísticas desde cualquier dispositivo en la misma red.

Hardware

    MCU: ESP32.

    Pantalla: OLED 128×64 (SSD1306).

    Controles: 5 pulsadores.

    Apmentación: 2 baterías de ptio en paralelo.

    Power: switch ON/OFF y módulo de carga.

Juegos incluidos

Snake, Tetris, Frogger, Pong, Breakout, Geometry Dash, Dino Chrome y Doodle Jump.
